# AI Development Workflow

Before generating or modifying code:

1. Read `PROJECT_CONTEXT.md`
2. Review the relevant documentation in this directory.
3. Follow `05-AI-Development-Rules.md`.
4. Use the templates from `06-Prompting-Guide.md`.
5. Update `07-Project-Progress.md` after completing major work.

---

# Documentation Principles

- Documentation is the single source of truth.
- Update documentation before implementing major architectural changes.
- Archive obsolete documents instead of deleting them.
- Keep implementation aligned with the Product Foundation and Technical Design.
